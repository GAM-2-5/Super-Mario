UPUTE ZA KORIŠTENJE CUSTOM ZUVČNIH EFEKATA

Ako želite promijeniti određene zvučne efekte u igri ili uvodnu špicu, 
željene datoteke morate staviti u 'Custom' folder po sljedećim pravilima.

Ako želite promjeniti: zvuk uvodne špice, imenujte file 'Opening' s nastavkom .mp3
                       video uvodne špice, imenujte file 'Super Mario Opening' s nastavkom .mp4
                       pozadinski zvuk igre, imenujte file 'Mario song' s nastavkom .mp3
		       pozadinu igre, imenujte file 'pozadina' s nastavkom .png
                       zvuk skoka, imenujte file 'Mario Jump' s nastavkom .wav
                       zvuk fireball-a, imenujte file 'fireball' s nastavkom .wav
                       zvuk kraja igre, imenujte file 'game over' s nastavkom .mp3

*Ako želite promjeniti zvuk i video uvodne špice, to moraju biti zasebni i odvojeni file-ovi (mp4 i mp3).*
*Python može puštati samo videe od 30 FPS-a!*                  
Ako nešto napravite krivo, python će učitati podešene zvukove.
